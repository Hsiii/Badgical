# Badgical

Animated SVG badge builder built with Next.js App Router.

## Install

```bash
bun install
```

## Develop

```bash
bun run dev
```

## Check

```bash
bun run check
```

This project includes `bunfig.toml` with `minimumReleaseAge = 604800`.
